## Exercícios

1. Explica o que é uma fórmula numa folha de cálculo. Indica dois exemplos de operações que podem ser feitas com fórmulas. (3 valores)

        Resposta: ...uma fórmula de cálculo é usada para fazer cálculos autómaticos numa folha de calculo, ex:cálcular médias

2. Define “referência relativa” e “referência absoluta”. Dá um exemplo de cada. (3 valores)

        Resposta: ...referência absoluta: não muda
   referência relativa: muda quando uma fórmula é copiada

4. Na célula D2 encontra-se a fórmula:

    =$B$2*C2

    Que tipo de referência é usada em B2? O que acontece à referência C2 quando a fórmula é copiada para D3? (2 valores)

        Resposta: ...B2 é uma referência absoluta. ao opiar a fórmula para D3, C2 passa a C3.

5. Para que serve a formatação condicional? Dá dois exemplos práticos. (3 valores)

        Resposta: ...A formatação condicional serve para destacar automaticamente dados importantes.
exemplos: negativos a vermelho.



6. Explica o que é um filtro de dados e dá um exemplo prático da sua utilização. (2 valores)

        Resposta: ...Um filtro de dados serve para mostrar apenas os dados que cumprem um critério.

7. Indica a diferença entre “ordenar” e “filtrar” dados. (2 valores)

        Resposta: a diferernça entre ordenar e filtrar dados são:
   Ordenar: muda a ordem dos dados (ex.: do maior para o menor).
Filtrar: esconde os dados que não interessam.

8. Para que serve um gráfico numa folha de cálculo? Indica um tipo de gráfico e uma situação em que possa ser utilizado. (3 valores)

        Resposta: ...Um gráfico serve para visualizar dados de forma mais clara. exemplo: Gráfico de colunas para comparar vendas mensais.

9. Escreve a fórmula necessária para calcular a prestação mensal de um empréstimo a 12 meses com taxa anual de 3% sobre 500€. Não é necessário calcular o valor final. (2 valores)

        Resposta: ...A fórmula para a prestação mensal é: =PGTO(3%/12;12;500)
